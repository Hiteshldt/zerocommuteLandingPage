# ZeroCommute — Landing Page Build Brief (Prompt for Junior Engineer)

## Goal & Tone

Build a **very simple, professional, and elegant website** for a climate-tech mobility solution. Prioritize clarity, credibility, and fast performance. Design must be **mobile-first**, minimal, and clean. Visual tone: modern, calm, climate-forward, with generous whitespace and short copy.

---

## Tech Stack (keep it lightweight)

* **Framework:** Next.js (App Router) + **TypeScript**
* **Styling:** Tailwind CSS (utility-first, no heavy UI kits)
* **Icons (optional):** lucide-react
* **Images:** place under `/public/images` and import with `next/image`
* **Logo:** Use the provided official logo file (place it in `/public/images/logo.png`)

> If you prefer pure static HTML/CSS, keep the same section structure and content below.

## Global Design System

* **Brand Colors:**

  * Dark Green `#28723e` (primary)
  * Light Grayish Beige `#c4d1bd` (background accent)
  * Soft Mint Green `#ceebcd` (secondary accent)
  * Muted Aqua Green `#68a68c` (supportive accent)
  * Deep Navy Blue `#002f4b` (for strong contrast / footer / header background)
* **Typography:** System fonts or Inter (400/600/800). Headings tight, body 16–18px.
* **Spacing:** Section paddings \~ `py-16 md:py-24`; max content width `max-w-6xl`.
* **Cards:** subtle shadows, rounded-2xl; hover lift on desktop.
* **Buttons:** Primary (solid Dark Green), Secondary (outline Navy Blue). Always high-contrast.
* **Accessibility:** semantic tags, alt text for every image, focus styles visible.
